## Overview

**Sabor Local** ("Local Flavor") is an early-stage project aimed at building a discovery platform for neighborhood restaurants — giving small local businesses a digital presence and allowing customers to browse menus, specials, and contact details without relying on third-party aggregators.

This repository is in its initial phase. Architecture and implementation are actively being planned.

---

## Planned Features

- Restaurant listing with location and category filters
- Digital menu pages per establishment
- Daily specials and highlights
- Mobile-first responsive design
- Simple onboarding flow for restaurant owners
